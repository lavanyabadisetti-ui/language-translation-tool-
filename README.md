🌍 Language Translation Tool

«Breaking language barriers through seamless communication.»

A modern web-based Language Translation Tool that enables users to translate text between multiple languages quickly and efficiently. Built using core web technologies and integrated with a translation API, this project demonstrates practical implementation of API consumption, asynchronous JavaScript, and responsive user interface design.

---

📖 Project Overview

The Language Translation Tool is designed to facilitate multilingual communication by allowing users to convert text from one language to another in real time. The application offers a clean and intuitive interface, making translation accessible and convenient for users across different linguistic backgrounds.

This project was developed to strengthen frontend development skills while gaining hands-on experience with REST APIs and dynamic content rendering.

⚙️ How the Application Works

1. User enters text in the input area.
2. User selects the source language.
3. User selects the target language.
4. JavaScript sends a request to the translation API using the Fetch API.
5. The API processes the request and returns the translated text.
6. The translated output is displayed instantly on the webpage.
